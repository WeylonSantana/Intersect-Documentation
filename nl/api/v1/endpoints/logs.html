# Logs

# Player Activity

Retrieve player activity

# Request

URL : /api/v1/logs/player/[lookupKey]/activity

Method : Character's id.

Lookup Key: : GET

Authorization Required : Yes

Content Type : NA

Body Empty

# Response

Condition : NA

Code : 200 SUCCESS

Example

{
  "Total": 9,
  "Page": 0,
  "PageSize": 5,
  "Count": 5,
